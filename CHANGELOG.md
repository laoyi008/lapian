# 更新日志

## 2025-12-03 - 视频拉片分析功能优化

### 新增功能

1. **Markdown表格展示**
   - 分镜分析结果以专业的Markdown表格形式展示
   - 表格包含：镜号、画面截图、景别/角度、运动方式、画面内容、时长(秒)

2. **音视频整合分析**
   - 音频识别结果自动整合到对应镜头的画面内容描述中
   - 语音内容作为画面描述的一部分，不再单独展示

3. **详细画面分析**
   - 每个镜头的画面内容描述更加详细
   - 包含场景环境、人物外貌、动作表情、物体道具、色彩光线、构图布局等
   - 描述达到能从文字反推画面的程度

4. **视频总结**
   - 自动生成视频的创意构思分析
   - 总结表现手法和镜头语言运用
   - 分析如何传达产品信息和核心内容

5. **简化AI提示词生成**
   - 生成的提示词更加简洁、直接可用
   - 整合所有镜头信息，不过度拆分
   - 可直接复制使用，无需二次编辑

### 技术改进

1. **音频时间轴匹配**
   - 音频识别结果按时间轴与镜头匹配
   - 每个镜头自动关联对应时间段的语音内容

2. **结构化数据提取**
   - 从AI分析结果中提取景别、角度、运动方式等结构化信息
   - 自动计算每个镜头的时长

3. **Markdown渲染优化**
   - 添加表格样式支持
   - 优化图片在表格中的显示效果
   - 改进整体排版和可读性

### 用户体验优化

- 分析流程更加清晰，进度提示更准确
- 表格形式展示更专业、更易读
- 一键复制功能支持完整的分析报告
- AI提示词生成更贴合实际使用需求
