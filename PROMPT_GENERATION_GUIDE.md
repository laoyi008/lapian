# 📝 AI视频提示词生成指南

## 🎯 功能概述

AI视频提示词生成功能基于视频拉片分析结果，自动生成包含**分时段分镜描述**和**字幕内容**的专业视频创作提示词。

---

## ✨ 核心特性

### 1. 分时段分镜描述

生成的提示词会按照时间顺序，详细描述每个镜头：

```
【镜头1】（0.0秒-5.0秒，时长5.0秒）
景别/角度：中景/平视
运动方式：固定镜头
画面内容：主持人站在产品前，面带微笑，背景简洁明亮
人物对白/旁白："大家好，今天给大家介绍一款新产品"

【镜头2】（5.0秒-8.0秒，时长3.0秒）
景别/角度：特写/平视
运动方式：推镜头
画面内容：产品特写，展示细节和质感
人物对白/旁白："这款产品采用了最新的技术"

【镜头3】（8.0秒-10.5秒，时长2.5秒）
景别/角度：中景/平视
运动方式：固定镜头
画面内容：主持人手持产品进行使用演示
人物对白/旁白："使用起来非常简单方便"
```

### 2. 结合字幕内容

如果视频中有人声，提示词会自动整合字幕内容：

- ✅ 每个镜头的对白或旁白
- ✅ 与画面内容的对应关系
- ✅ 语音的时间点和内容

### 3. 专业的视频描述

生成的提示词包含：

- **时间信息**：每个镜头的起止时间和时长
- **拍摄手法**：景别、角度、镜头运动
- **画面内容**：详细的视觉元素描述
- **人物对白**：如果有字幕，会标注人物说话内容
- **整体风格**：色调、氛围、节奏等

---

## 🚀 使用步骤

### 第1步：完成视频分析

1. 上传视频文件
2. 点击"开始分析"
3. 等待分析完成
4. 确认"视频分镜拆解分析"表格已显示

### 第2步：生成提示词

1. 滚动到"AI视频提示词生成"卡片
2. 点击"生成提示词"按钮
3. 等待AI生成（使用百度NLP技术）
4. 查看生成的提示词

### 第3步：使用提示词

1. 点击"复制"按钮
2. 粘贴到AI视频生成工具中
3. 根据需要进行微调
4. 生成视频

## 🎨 提示词包含的要素

### 1. 时间信息 ⏱️

每个镜头都标注了：
- 起始时间（X秒）
- 结束时间（Y秒）
- 时长（Z秒）

**示例：**
```
【镜头1】（0.0秒-5.0秒，时长5.0秒）
```

### 2. 拍摄手法 🎬

包含专业的拍摄术语：
- **景别**：远景、全景、中景、近景、特写
- **角度**：平视、俯视、仰视、侧面
- **运动**：固定镜头、推镜头、拉镜头、摇镜头、移镜头

**示例：**
```
景别/角度：中景/平视
运动方式：固定镜头
```

### 3. 画面内容 🖼️

详细描述画面中的元素：
- 主体人物或物体
- 背景环境
- 光线和色彩
- 动作和姿态
- 细节和质感

**示例：**
```
画面内容：主持人站在产品展示台前，面带微笑，身着职业装。
背景是简洁的白色墙面，光线明亮均匀。产品摆放在展示台上，清晰可见。
```

### 4. 人物对白 💬

如果有字幕，会标注：
- 人物说话的内容
- 旁白或解说词
- 与画面的对应关系

**示例：**
```
人物对白/旁白："大家好，今天给大家介绍一款新产品"
```

### 5. 视觉风格 🎨

描述整体的视觉感受：
- 色调（明亮、暗沉、温暖、冷峻）
- 氛围（专业、轻松、严肃、活泼）
- 质感（清晰、柔和、锐利、模糊）

**示例：**
```
画面风格：专业、清晰、明亮
```

## 💡 使用技巧

### 技巧1：确保字幕识别成功

在生成提示词前，确认：
- ✅ 音频识别完成
- ✅ "音频识别调试信息"卡片显示字幕内容
- ✅ 分镜表格的"字幕内容"列有文字

**如果没有字幕：**
- 生成的提示词不会包含人物对白
- 只包含画面描述和拍摄手法

### 技巧2：检查分镜数据完整性

确认分镜拆解表格包含：
- ✅ 所有镜头的截图
- ✅ 景别/角度信息
- ✅ 运动方式
- ✅ 画面内容描述
- ✅ 字幕内容（如果有）

### 技巧3：多次生成对比

如果第一次生成的提示词不满意：
1. 点击"重新生成"按钮
2. AI会生成不同风格的提示词
3. 选择最符合需求的版本

### 技巧4：手动优化提示词

生成后可以手动调整：
- 修改描述的详细程度
- 调整语言风格
- 添加特定的要求
- 删除不需要的部分

### 技巧5：保存常用提示词

对于满意的提示词：
1. 复制到文本文件保存
2. 建立自己的提示词库
3. 作为模板重复使用

## 🔧 常见问题

### 问题1：提示词没有包含字幕内容

**原因：**
- 音频识别失败
- 字幕数据为空
- 视频没有人声

**解决方案：**
1. 查看"音频识别调试信息"卡片
2. 确认字幕是否识别成功
3. 如果没有字幕，提示词只包含画面描述

### 问题2：提示词格式混乱

**原因：**
- AI生成时的随机性
- 输入数据不完整

**解决方案：**
1. 点击"重新生成"
2. 确保分镜数据完整
3. 手动调整格式

### 问题3：提示词过于简单

**原因：**
- 画面内容描述不够详细
- AI生成的简化版本

**解决方案：**
1. 重新生成，尝试获得更详细的版本
2. 手动补充细节描述
3. 参考示例提示词的格式

### 问题4：提示词过于冗长

**原因：**
- AI生成了过于详细的描述
- 包含了不必要的信息

**解决方案：**
1. 手动删减不必要的部分
2. 保留核心信息
3. 简化语言表达

### 问题5：时间信息不准确

**原因：**
- 分镜切换检测不准确
- 时间戳计算有误

**解决方案：**
1. 查看分镜拆解表格的时间信息
2. 手动调整提示词中的时间
3. 参考实际视频的时间点
