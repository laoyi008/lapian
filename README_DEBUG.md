## 🎯 如果还是卡在30%

### 1. 查看控制台最后一条日志
- 如果是"开始分析 X 个镜头" → 正在处理中，请等待
- 如果是"HTTP错误" → 网络问题，检查连接
- 如果是"任务处理中" → API处理慢，继续等待

### 2. 常见问题
| 问题 | 解决方案 |
|------|----------|
| 一直显示"轮询尝试" | 正常，等待API处理（最多60秒） |
| 显示HTTP错误 | 检查网络，刷新页面重试 |
| 显示API错误 | 查看具体错误信息，可能是视频格式问题 |
