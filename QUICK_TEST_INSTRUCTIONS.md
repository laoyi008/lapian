# 🚀 快速测试说明

## 🎯 目的

快速测试百度语音识别API是否正常工作，诊断字幕识别问题。

---

## ⚡ 3步快速测试

### 第1步：访问测试工具

在浏览器地址栏输入：`/audio-test`

或点击导航栏的"语音识别测试"

### 第2步：录音测试

1. 按 `F12` 打开控制台
2. 点击"开始录音"
3. 清晰地说："你好，这是一段测试语音"
4. 点击"停止录音"

### 第3步：查看结果

**✅ 成功标志：**

页面显示：
```
识别结果
你好，这是一段测试语音
```

控制台显示：
```
✅ 识别成功: 你好，这是一段测试语音
```

**❌ 失败标志：**

页面显示：
```
识别失败: speech quality problem (err_no: 3301)
```

---

## 🔍 快速诊断

### 情况1：测试工具识别成功 ✅

**说明：** API配置正确，问题在视频拉片分析

**下一步：**
1. 检查视频文件是否有音频
2. 查看视频拉片分析的控制台日志
3. 确认音频提取是否成功

### 情况2：测试工具识别失败 ❌

**说明：** API本身有问题

**检查项：**

#### err_no = 3301（音频质量问题）
- 重新录音，确保清晰说话
- 提高音量
- 在安静环境中测试

#### err_no = 3302（鉴权失败）
- 检查 `.env` 文件中的 `VITE_APP_ID`
- 确认API配置正确

#### 其他错误
- 查看完整的err_msg
- 查看控制台详细日志
- 参考 [AUDIO_TEST_GUIDE.md](AUDIO_TEST_GUIDE.md)

---

|---------|------------|---------|
| ✅ 成功 | ❌ 失败 | 视频音频提取或处理 |
| ❌ 失败 | ❌ 失败 | API配置或音频格式 |

## 💡 测试建议

- ✅ 清晰发音
- ✅ 标准普通话
- ✅ 安静环境
