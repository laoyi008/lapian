## ✨ 功能特性

### 🎥 智能分镜分析
- 自动提取视频关键帧
- AI深度分析画面内容
- 识别场景、人物、动作
- 分析拍摄手法和构图

### 🎤 音频识别
- 提取视频音频轨道
- 智能语音转文字
- 支持中文普通话
- 整合到分析结果

### 🤖 AI提示词生成
- 基于视频分析结果
- 使用百度NLP技术
- 流式输出，实时显示
- 一键复制，即用即走

### 📊 可视化展示
- 清晰的分析报告
- Markdown格式支持
- 表格化数据展示
- 支持导出功能

## 🚀 快速开始

### 环境要求
- Node.js 16+
- pnpm/npm
- 现代浏览器（Chrome 90+）

### 安装依赖
```bash
pnpm install
```

### 配置环境变量
```bash
# .env
VITE_APP_ID=your_app_id
VITE_API_ENV=production
```

### 启动开发服务器
```bash
pnpm run dev
```

### 构建生产版本
```bash
pnpm run build
```

### 代码检查
```bash
pnpm run lint
```

## 🔍 调试技巧

### 查看详细日志
```
1. 按 F12 打开开发者工具
2. 切换到 Console 标签
3. 上传视频并开始分析
4. 观察详细的日志输出
```

### 常见问题
| 问题 | 解决方案 |
|------|----------|
| 进度卡住 | 查看控制台日志 |
| 分析失败 | 检查网络连接 |
| 无法上传 | 检查视频格式 |
| 提示词空白 | 重新生成 |
