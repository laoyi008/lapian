# 项目总结

## 📋 项目信息

**项目名称：** 短视频拉片分析工具  
**项目类型：** Web应用  
**技术栈：** React + TypeScript + Tailwind CSS + shadcn/ui  
**AI服务：** 百度智能云  
**版本：** v2.0  
**状态：** ✅ 已完成并优化

---

## 🎯 项目目标

打造一个专业的短视频分析与创作辅助平台，帮助用户：
1. 深度拆解短视频内容
2. 分析视频的拍摄手法和视觉元素
3. 识别视频中的语音内容
4. 生成专业的AI视频生成提示词

---

## ✨ 核心功能

### 1. 视频上传 ✅
- 支持常见视频格式（MP4、MOV等）
- 文件大小验证
- 拖拽上传支持
- 实时预览

### 2. 智能分镜分析 ✅
- 自动提取关键帧
- 使用百度图像内容理解API
- 识别场景、人物、动作
- 分析拍摄手法和构图

### 3. 音频识别 ✅
- 提取视频音频轨道
- 使用百度短语音识别API
- 转换语音为文字
- 整合到分析结果中

### 4. AI提示词生成 ✅
- 基于视频分析结果
- 使用百度NLP技术
- 流式输出，实时显示
- 一键复制功能

### 5. 分析结果展示 ✅
- 清晰的视觉呈现
- Markdown格式支持
- 表格化展示
- 支持导出

### 百度智能云集成

#### 图像内容理解API
- **功能：** 分析视频画面内容
- **端点：** `/rest/2.0/image-classify/v1/image-understanding/`
- **特点：** 异步处理，轮询获取结果

#### 短语音识别API
- **功能：** 识别视频音频内容
- **端点：** `/rest/2.0/speech/v1/asr`
- **特点：** 支持中文普通话，16000Hz采样率

#### 自然语言处理NLP API
- **功能：** 生成AI视频提示词
- **端点：** `/v2/chat/completions`
- **特点：** 流式输出，实时显示
