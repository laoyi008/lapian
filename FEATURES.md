# 短视频拉片分析工具 - 功能说明

## 核心功能

### 1. 视频上传与预处理
- 支持常见视频格式（MP4、MOV等）
- 自动提取视频时长和关键帧
- 智能分镜，提取代表性画面

### 2. 画面分析
使用百度智能云图像内容理解API，对每个镜头进行深度分析：

#### 分析维度
- **景别**：特写、近景、中景、全景、远景
- **角度**：平视、俯视、仰视等
- **运动方式**：固定镜头、推镜头、拉镜头、摇镜头、移镜头、跟镜头等
- **画面内容**：
  - 场景环境和背景
  - 人物外貌、服装、位置
  - 人物动作和表情
  - 物体和道具
  - 色彩和光线
  - 构图和布局

### 3. 音频识别
使用百度智能云短语音识别API：
- 提取视频音频轨道
- 识别语音内容
- 按时间轴匹配到对应镜头
- 自动整合到画面内容描述中

### 4. 分镜拆解报告
以专业的Markdown表格形式展示：

| 镜号 | 画面截图 | 景别/角度 | 运动方式 | 画面内容 | 时长(秒) |
|------|----------|-----------|----------|----------|----------|
| 1    | 截图     | 中景/平视 | 固定镜头 | 详细描述... | 3.5 |

### 5. 视频总结
自动生成三个维度的总结：
- **创意构思**：分析视频的整体创意和叙事结构
- **表现手法**：总结使用的镜头语言和拍摄技巧
- **信息传达**：评估如何有效传达核心信息

### 6. AI提示词生成
使用百度文心大模型：
- 基于完整的分镜分析
- 整合画面和音频信息
- 生成简洁、直接可用的提示词
- 适合各类AI视频生成工具使用

## 使用场景

1. **视频创作者**
   - 学习优秀视频的拍摄手法
   - 分析镜头语言运用
   - 获取创作灵感

2. **视频编辑**
   - 快速了解视频结构
   - 辅助剪辑决策
   - 生成分镜脚本

3. **AI视频生成**
   - 从现有视频提取提示词
   - 学习视频描述方法
   - 优化AI生成效果

4. **视频教学**
   - 教学案例分析
   - 镜头语言讲解
   - 视频制作培训

## 输出格式

### 分镜拆解表格
- Markdown格式，易于复制和分享
- 包含截图、文字描述和技术参数
- 支持导出和打印

### AI提示词
- 纯文本格式
- 直接可用，无需修改
- 适配主流AI视频生成工具

## 技术栈

- **前端框架**：React + TypeScript
- **UI组件**：shadcn/ui + Tailwind CSS
- **图像分析**：百度智能云图像内容理解API
- **语音识别**：百度智能云短语音识别API
- **AI生成**：百度文心大模型
- **音频处理**：Web Audio API
