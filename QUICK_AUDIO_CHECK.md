# 快速音频识别检查

## 🎯 问题：字幕内容没有提取出来

---

## ⚡ 快速诊断（3步）

### 第1步：打开控制台

按 `F12` → 切换到 **Console** 标签

### 第2步：上传视频并分析

上传视频 → 点击"开始分析" → 观察控制台

### 第3步：查找关键信息

在控制台中搜索以下关键词：

---

## 🔍 关键检查点

### ✅ 检查点1：音频解码

**搜索：** `音频解码成功`

**看到这个 = 成功：**
```
✅ 音频解码成功！
- 音频时长: 120 秒
```

**看到这个 = 失败：**
```
❌ 视频没有音频轨道或音频解析失败
```
→ **原因：** 视频没有音频  
→ **解决：** 使用包含音频的视频

---

### ✅ 检查点2：识别结果

**搜索：** `识别成功`

**看到这个 = 成功：**
```
✅ 第 1 段识别成功！
- 识别文本: "大家好，今天给大家介绍..."
```

**看到这个 = 失败：**
```
⚠️ 第 1 段音频无识别结果
- err_no: 3301
- err_msg: speech quality problem
```
→ **原因：** 音频质量问题或无人声  
→ **解决：** 使用人声清晰的视频

---

### ✅ 检查点3：最终统计

**搜索：** `音频识别完成`

**看到这个 = 成功：**
```
=== 音频识别完成 ===
识别出的字幕段数: 2
字幕内容: {
  "0": "大家好...",
  "60": "接下来..."
}
```

**看到这个 = 失败：**
```
识别出的字幕段数: 0
⚠️ 未识别到任何语音内容
```
→ **原因：** 所有音频段都没有识别出内容  
→ **解决：** 检查视频是否有清晰人声

---

## 🎬 推荐测试视频

### ✅ 好的测试视频

- 有清晰的人声对白或解说
- 音量适中
- 背景音乐不要过大
- 标准普通话
- 1-3分钟时长

### ❌ 不适合的测试视频

- 纯音乐视频（无人声）
- 背景音乐过大
- 音量过小
- 方言或外语
- 噪音过大

---

## 🔧 常见错误代码

| err_no | 含义 | 解决方案 |
|--------|------|----------|
| 0 | 成功 | ✅ 正常 |
| 3301 | 音频质量问题 | 使用人声清晰的视频 |
| 3302 | 鉴权失败 | 检查API配置 |
| 3303 | 服务器问题 | 稍后重试 |
| 3304 | QPS超限 | 降低请求频率 |
| 3305 | 日请求量超限 | 明天再试 |

---

## 💡 快速解决方案

### 问题：err_no = 3301

**原因：** 音频质量问题

**快速检查：**
1. 播放视频，确认有人声
2. 检查音量是否正常
3. 确认不是纯音乐

**解决方案：**
- 使用人声清晰的视频
- 调高音频音量
- 降低背景音乐

### 问题：识别出的字幕段数为0

**原因：** 所有音频段都没有识别出内容

**快速检查：**
1. 查看每段的err_no
2. 确认视频有人声
3. 检查音频质量

**解决方案：**
- 使用标准普通话视频
- 确保人声清晰
- 避免背景噪音
