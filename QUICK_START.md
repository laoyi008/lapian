# 卡密系统快速开始指南

## 🚀 快速测试流程

### 步骤1：管理员生成卡密

1. 使用管理员账号登录系统
2. 访问管理后台（导航栏 → 管理后台）
3. 点击"卡密管理"标签
4. 在"生成卡密"区域：
   - 积分数量：输入 `1000`
   - 生成数量：输入 `1`
5. 点击"生成卡密"按钮
6. 在卡密列表中找到刚生成的卡密
7. 点击复制按钮，复制卡密码

### 步骤2：用户兑换卡密

1. 退出管理员账号，使用普通用户账号登录（或注册新账号）
2. 访问积分充值页面（导航栏 → 积分充值）
3. 点击"卡密兑换"标签
4. 在输入框中粘贴刚才复制的卡密码
5. 点击"立即兑换"按钮（或按回车键）
6. 看到成功提示：`兑换成功！获得 1000 积分`
7. 查看页面顶部的积分余额，确认积分已增加

### 步骤3：验证卡密状态

1. 切换回管理员账号
2. 进入管理后台 → 卡密管理
3. 在卡密列表中找到刚才使用的卡密
4. 确认状态已变为"已使用"
5. 确认"使用者"列显示了用户名
6. 确认"使用时间"已记录

### 步骤4：测试重复兑换

1. 使用同一个卡密再次尝试兑换
2. 应该看到错误提示：`卡密不存在或已被使用`
3. 确认积分没有增加

## 🎯 关键功能点

### 卡密格式
```
XXXX-XXXX-XXXX-XXXX
例如：A2B3-C4D5-E6F7-G8H9
```

### 生成规则
- 使用字符：A-Z（排除I、O）和2-9（排除0、1）
- 总长度：16个字符 + 3个连字符
- 自动生成，保证唯一性

### 使用限制
- 每个卡密只能使用一次
- 必须登录才能兑换
- 卡密不区分大小写

## 💡 使用技巧

### 管理员
1. **批量生成**：可以一次生成多个相同积分的卡密（最多100个）
2. **快速复制**：点击卡密列表中的复制按钮快速复制
3. **筛选查看**：可以按状态筛选卡密（未使用/已使用）

### 用户
1. **快速兑换**：输入卡密后按回车键快速兑换
2. **自动转换**：输入的卡密会自动转换为大写
3. **查看历史**：在"我的"页面可以查看兑换记录

## 🔍 常见问题排查

### 问题1：生成卡密失败
**可能原因**：
- 不是管理员账号
- 积分数量或生成数量无效
- 网络连接问题

**解决方法**：
- 确认使用管理员账号登录
- 检查输入的数值是否有效
- 刷新页面重试

### 问题2：兑换卡密失败
**可能原因**：
- 卡密已被使用
- 卡密不存在
- 卡密格式错误
- 未登录

**解决方法**：
- 确认卡密是否正确
- 检查是否已登录
- 联系管理员确认卡密状态

### 问题3：积分没有增加
**可能原因**：
- 兑换未成功
- 页面未刷新

**解决方法**：
- 查看是否有成功提示
- 刷新页面查看最新积分
- 查看积分交易记录
